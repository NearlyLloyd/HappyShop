# HappyShop

JavaFX project designed to challenge students to fix+maintain the project.

##  Setup For Developers

After opening the project in IntelliJ:

- Go to the `systemSetup` package.
- Run `SetOrderFileSystem` and `SetDatabase` **once only** to set up initial files and database (This clears all previous data).

## Running the System

- Simply RUN `Launcher` for the system to boot correctly.
